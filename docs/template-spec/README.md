# Template Specification Examples

This directory contains the complete specification that was used to build the
Open Specification Template for Python itself.

These files serve as reference examples showing what filled-in specification
documents look like. Use them for inspiration when writing your own specs.

## Contents

| File | Purpose |
|------|---------|
| [PRODUCT.md](PRODUCT.md) | What the template is and why it exists |
| [ARCHITECTURE.md](ARCHITECTURE.md) | How the template is structured |
| [SPECIFICATION.md](SPECIFICATION.md) | Detailed requirements for template behavior |
| [ROADMAP.md](ROADMAP.md) | Phased plan for building the template |

## How to Use These

1. Open the document type you're working on (e.g., `PRODUCT.md`)
2. Read through the example to see the structure and level of detail
3. Return to `spec/PRODUCT.md` and write your own version

These are **examples only** — your specification should describe your project,
not this template.
